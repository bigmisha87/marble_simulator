# Marble Simulator — Galton Board

A browser-based marble sorter / Galton board physics simulator, built with [Matter.js](https://brm.io/matter-js/). Drop up to **10,000** marbles through a fully configurable peg board and watch them sort into bins.

## 🎮 Play

### 👉 https://bigmisha87.github.io/marble_simulator/

Runs in any modern browser — no install needed.

## Features

- **Marbles** — count (up to 10K), size, random seed, and adjustable reservoir height.
- **Board** — bin count & height, peg rows and pegs-per-row (up to 50 each), peg size, and board width.
- **Physics** — gravity, marble/peg bounce, surface grip, air drag, simulation speed, and anti-stuck.
- **Gate & layout** — peg shape (circle / diamond / random), peg layout (grid / zigzag / random), and shelf opening style.
- Presets, live bin statistics, pause/resume, focus view, and batch adding on the fly.
